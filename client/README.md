# Secure - Client

Securely upload and share files.

## Installation

For development

1. `git clone https://github.com/Luca-Castelnuovo/Secure.git`
2. `cd client`
3. `yarn install`
4. Start development server `yarn start`

For production

1. `git clone https://github.com/Luca-Castelnuovo/Secure.git`
2. `cd client`
3. `yarn install`
4. `yarn build`
